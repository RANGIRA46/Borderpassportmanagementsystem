## 🎓 **Usage Guide**

### **For Developers**:

#### **Using the Wizard Component**:
```tsx
import { WizardSteps } from './components/modern/WizardSteps';

<WizardSteps
  steps={[
    { id: 'step1', title: 'Step 1', description: 'Description' },
    { id: 'step2', title: 'Step 2', description: 'Description' }
  ]}
  currentStep={currentStep}
  onNext={() => setCurrentStep(prev => prev + 1)}
  onBack={() => setCurrentStep(prev => prev - 1)}
  onSubmit={handleSubmit}
>
  {renderStepContent()}
</WizardSteps>
```

#### **Using Service Cards**:
```tsx
import { ServiceCard } from './components/modern/ServiceCard';

<ServiceCard
  title="Service Name"
  description="Service description"
  icon={<Icon />}
  color="from-navy-medium to-blue-medium"
  estimated="3-5 days"
  popular={true}
  onClick={() => navigate('service')}
/>
```

#### **Using the Language Switcher**:
```tsx
import { FloatingLanguageSwitcher } from './components/modern/FloatingLanguageSwitcher';

// Add to layout (already integrated in App.tsx)
{!isAdminUser && <FloatingLanguageSwitcher />}
```

### **For Designers**:
- All color values in `/styles/globals.css`
- Typography scale follows Tailwind defaults
- Spacing uses 8px grid system
- Gradients use CSS variables
- Icons from Lucide React library
